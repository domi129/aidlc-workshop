# 테이블 오더 시스템 기능 명세서

## 📱 고객 페이지 기능

### 1. 로그인/인증
- 테이블 번호, 매장 번호, 비밀번호로 로그인
- 로그아웃 기능
- 테이블별 세션 관리
- 로그인 후 테이블 정보 헤더 표시

### 2. 메뉴 주문
- **카테고리 필터링**
  - 전체, 메인, 사이드, 음료 탭으로 분류
  - 탭 클릭으로 카테고리별 메뉴 필터링
- **메뉴 표시**
  - 메뉴 이미지 표시
  - 메뉴명, 가격, 카테고리 정보
  - 그리드 레이아웃
- **장바구니 기능**
  - 메뉴 클릭으로 장바구니 담기
  - 수량 조절 (+/- 버튼)
  - 실시간 총액 계산
  - 장바구니 비우기
- **주문하기**
  - 장바구니 내역 확인
  - 주문 완료 알림

### 3. 주문 내역 관리
- **실시간 조회**
  - 내 주문 내역 자동 새로고침 (3초마다)
  - 테이블별 주문만 표시
- **주문 상태**
  - 준비중 (주황색 배지)
  - 완료 (초록색 배지)
- **주문 정보**
  - 주문 번호
  - 주문 시간
  - 메뉴별 수량 및 금액
  - 주문별 총액
- **전체 합계**
  - 모든 주문의 합계 금액 표시

### 4. 결제
- **결제 수단 선택 모달**
  - 현금 결제 버튼 (💵)
  - 카드 결제 버튼 (💳)
  - 총 결제 금액 표시
- **현금 결제**
  - 자동 직원 호출
  - "직원이 방문합니다" 안내
- **카드 결제**
  - 즉시 결제 처리
  - 결제 완료 알림
- **결제 후 처리**
  - 주문 내역 초기화
  - 장바구니 유지

### 5. 직원 호출
- **호출 버튼**
  - 헤더에 "🔔 직원 호출" 버튼 배치
- **요청 사항 선택 모달**
  - 물 💧
  - 수저 🥢
  - 휴지 🧻
  - 밥 추가 🍚
  - 반찬 추가 🌶️
  - 테이블 정리 🧹
  - 문의사항 ❓
  - 기타 🔔
- **다중 선택**
  - 여러 항목 동시 선택 가능
  - 선택 개수 표시
  - 선택된 항목 강조 표시
- **호출 완료**
  - 직원 호출 알림
  - 선택 항목 초기화

## 👨‍💼 관리자 페이지 기능

### 1. 대시보드 통계
- **실시간 지표**
  - 대기 주문 수
  - 완료 주문 수
  - 오늘 결제 건수
  - 오늘 매출 (원 단위)
  - 직원 호출 건수 (대기중 강조)
- **자동 업데이트**
  - 3초마다 자동 갱신

### 2. 직원 호출 관리 탭 (🔔)
- **대기중 호출**
  - 테이블 번호
  - 호출 사유 (선택된 요청 사항)
  - 호출 시간
  - 완료 처리 버튼
  - 주황색 테두리 강조
- **완료된 호출**
  - 호출 시간 및 완료 시간
  - 삭제 버튼
  - 회색 테두리
- **실시간 업데이트**
  - 3초마다 자동 갱신
- **알림**
  - 대기중 호출 건수 배지 표시

### 3. 주문 관리 탭 (📋)
- **대기중 주문**
  - 테이블별 그룹화
  - 주문 번호
  - 주문 시간
  - 메뉴별 상세 내역
  - 주문 총액
  - 완료 처리 버튼 (✓)
  - 주황색 테두리
- **완료된 주문**
  - 테이블별 그룹화
  - 대기중으로 변경 버튼 (↺)
  - 초록색 테두리
  - 투명도 적용
- **실시간 업데이트**
  - 3초마다 자동 갱신
- **알림**
  - 대기중 주문 건수 배지 표시

### 4. 결제 내역 탭 (💰)
- **오늘의 정산 대시보드**
  - 그라데이션 배경 (보라색)
  - 총 매출
  - 현금 매출
  - 카드 매출
  - 결제 건수
- **날짜별 결제 내역**
  - 날짜별 그룹화 및 정렬 (최신순)
  - 날짜별 합계 및 건수
- **결제 상세 정보**
  - 테이블 번호
  - 결제 수단 배지 (현금: 초록색, 카드: 파란색)
  - 결제 시간
  - 결제 금액
  - 주문 상세 내역
  - 결제 취소 버튼
- **결제 취소 기능**
  - 확인 팝업
  - 주문 복원 (완료 상태로)
  - 결제 내역 삭제
- **실시간 업데이트**
  - 5초마다 자동 갱신

### 5. 통계 분석 탭 (📊)

#### 주요 지표 카드
- **오늘 매출** (파란색 아이콘)
  - 총 매출 금액
- **결제 건수** (초록색 아이콘)
  - 오늘 결제 건수
- **평균 객단가** (주황색 아이콘)
  - 결제당 평균 금액
- **인기 메뉴** (보라색 아이콘)
  - 가장 많이 팔린 메뉴

#### 시간대별 매출 차트
- **막대 그래프**
  - X축: 시간 (0시~23시)
  - Y축: 매출 금액
  - 오늘 하루 시간대별 매출
- **활용**
  - 피크 타임 파악
  - 인력 배치 최적화

#### 일별 매출 추이 차트
- **선 그래프**
  - X축: 날짜 (최근 7일)
  - Y축: 매출 금액
  - 총 매출 (파란색)
  - 현금 매출 (초록색)
  - 카드 매출 (주황색)
- **활용**
  - 매출 트렌드 분석
  - 요일별 패턴 파악

#### 결제 수단별 비율
- **파이 차트**
  - 현금 (초록색)
  - 카드 (파란색)
  - 금액 및 건수 표시
- **활용**
  - 결제 수단 선호도 파악
  - 현금 준비 계획

#### 테이블별 매출
- **가로 막대 그래프**
  - Y축: 테이블 번호
  - X축: 매출 금액
  - 오늘 테이블별 매출
- **활용**
  - 테이블 회전율 분석
  - 인기 테이블 파악

#### 메뉴별 판매 통계 테이블
- **데이터 테이블**
  - 순위
  - 메뉴명
  - 판매량 (개수)
  - 매출 (금액)
  - 비율 (진행 바 + 퍼센트)
- **전체 기간 데이터**
- **색상 코딩**
  - 순위별 다른 색상
- **활용**
  - 인기 메뉴 파악
  - 메뉴 구성 최적화
  - 재고 관리

### 6. 메뉴 관리 탭 (🍽️)
- **메뉴 추가**
  - 메뉴명 입력
  - 가격 입력 (숫자)
  - 카테고리 입력
  - 이미지 URL 입력 (선택사항)
  - 추가 버튼
- **카테고리별 그룹화**
  - 카테고리별 섹션 분리
  - 카테고리 제목 표시
- **메뉴 카드**
  - 메뉴 이미지 미리보기
  - 메뉴명
  - 가격
  - 수정/삭제 버튼
- **메뉴 수정**
  - 인라인 편집 모드
  - 모든 필드 수정 가능
  - 저장/취소 버튼
- **메뉴 삭제**
  - 확인 팝업
  - 즉시 삭제
- **이미지 처리**
  - 기본 이미지 제공
  - 로딩 실패 시 대체 이미지

### 7. 테이블 관리 탭 (🪑)
- **테이블 추가**
  - 테이블 번호 입력 (숫자)
  - 매장 번호 입력
  - 비밀번호 입력
  - 추가 버튼
- **테이블 목록**
  - 카드 형식 표시
  - 테이블 번호
  - 매장 번호
  - 비밀번호 숨김 (****)
- **테이블 수정**
  - 인라인 편집 모드
  - 테이블 번호 변경
  - 매장 번호 변경
  - 비밀번호 재설정
  - 저장/취소 버튼
- **테이블 삭제**
  - 확인 팝업
  - 즉시 삭제

## 🔧 기술 스택

### Frontend
- **React 18**
  - 함수형 컴포넌트
  - Hooks (useState, useEffect)
- **React Router v6**
  - 페이지 라우팅
  - 고객/관리자 페이지 분리
- **Recharts 2.10**
  - 막대 그래프 (BarChart)
  - 선 그래프 (LineChart)
  - 파이 차트 (PieChart)
  - 반응형 차트 (ResponsiveContainer)
- **CSS**
  - 커스텀 스타일링
  - 그리드 레이아웃
  - Flexbox
  - 모달 오버레이
  - 애니메이션 효과

### Backend
- **Node.js + Express**
  - RESTful API
  - JSON 응답
  - CORS 설정
- **데이터 저장**
  - 메모리 기반 (In-Memory)
  - orders 배열
  - payments 배열
  - tables 배열
  - staffCalls 배열
  - menus 배열

### API 엔드포인트

#### 인증
- `POST /api/tables/login` - 테이블 로그인

#### 메뉴
- `GET /api/menus` - 메뉴 목록 조회
- `POST /api/menus` - 메뉴 추가
- `PUT /api/menus/:id` - 메뉴 수정
- `DELETE /api/menus/:id` - 메뉴 삭제

#### 주문
- `GET /api/orders` - 모든 주문 조회
- `GET /api/orders/table/:tableNumber` - 테이블별 주문 조회
- `POST /api/orders` - 주문 생성
- `PUT /api/orders/:id/status` - 주문 상태 변경

#### 결제
- `GET /api/payments` - 결제 내역 조회
- `GET /api/payments/daily` - 일별 정산 조회
- `POST /api/tables/:tableNumber/payment` - 결제 처리
- `POST /api/payments/:id/cancel` - 결제 취소

#### 직원 호출
- `GET /api/staff-calls` - 호출 목록 조회
- `POST /api/staff-calls` - 직원 호출
- `PUT /api/staff-calls/:id/complete` - 호출 완료 처리
- `DELETE /api/staff-calls/:id` - 호출 삭제

#### 테이블
- `GET /api/tables` - 테이블 목록 조회
- `POST /api/tables` - 테이블 추가
- `PUT /api/tables/:id` - 테이블 수정
- `DELETE /api/tables/:id` - 테이블 삭제

## 📊 데이터 구조

### Menu
```javascript
{
  id: number,
  name: string,
  price: number,
  category: string,
  image: string
}
```

### Order
```javascript
{
  id: number,
  tableNumber: number,
  items: [
    {
      id: number,
      name: string,
      price: number,
      quantity: number
    }
  ],
  status: 'pending' | 'completed',
  createdAt: string (ISO 8601)
}
```

### Payment
```javascript
{
  id: number,
  tableNumber: number,
  orders: Order[],
  totalAmount: number,
  paymentMethod: '현금' | '카드',
  paymentDate: string (ISO 8601),
  status: 'paid'
}
```

### Table
```javascript
{
  id: number,
  tableNumber: number,
  storeNumber: string,
  password: string
}
```

### StaffCall
```javascript
{
  id: number,
  tableNumber: number,
  reason: string,
  status: 'pending' | 'completed',
  createdAt: string (ISO 8601),
  completedAt?: string (ISO 8601)
}
```
